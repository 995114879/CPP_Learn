## C++编程语言 01

### 一、 C++ 与 Python 对比 ...

- **速度和效率：**
    - C++： 编译型语言，执行速度快，效率高，更接近机器语言，常用于对性能要求高的领域，如游戏开发、操作系统等。
    - Python： 解释型语言，执行速度相对较慢，但开发效率高，语法简洁，常用于数据分析、人工智能等领域。

- **为什么人工智能班级学习C++？**
    - 深入理解底层原理，为算法优化提供基础。
    - 部分高性能库和框架使用C++开发，学习C++有助于更好地使用和理解这些工具。
    - 提升编程技能和代码效率。

### 二、 C++ 的发展与平台 ...

- **Windows 和 Linux 对比:**
    - Windows： 商业操作系统，拥有丰富的软件生态，但部分底层API与Linux不兼容。
    - Linux： 开源操作系统，底层更加透明，更受开发者欢迎，但用户界面和软件生态相对较弱。
- **C++ 的发展历程:**
    - 机器语言:  用二进制代码表示指令，例如 `0001101000`，难以理解和编写。
    - 汇编语言:  用助记符代替二进制代码，例如 `mov`, `jmp`, `add`，但仍然比较底层。
    - 高级语言 (C语言):   更接近人类语言，更容易理解和编写，但执行效率相对较低。
    - C++:  在C语言基础上发展而来，增加了面向对象特性，兼具开发效率和执行效率。
- **C++版本:**
    - C++98/03，C++【11】, C++14, C++17, C++【20】……  新版本不断引入新特性和改进，建议学习最新稳定版本。

### 三、 C++编译原理 ..

- Python：解释型语言，逐行解释执行代码。
- C++：编译型语言，需要先将代码编译成机器语言才能执行。

  **编译过程:**
   1. 预处理： 处理预处理指令，如 #include, #define 等。
   2. 编译： 将代码转换为汇编语言。
   3. 汇编： 将汇编语言转换为机器语言，生成目标文件。
   4. 链接： 将多个目标文件和库文件链接成可执行文件。

### 五、 第一个 C++ 程序 ....

```c++
#include <iostream> // 头文件，引入输入输出流

using namespace std; // 名词空间，避免命名冲突

int main() { // 主函数，程序的入口
  cout << "Hello, world!" << endl; // 打印语句
  return 0; // 返回值，表示程序正常结束
}
```
**代码解释:**
- `#include <iostream>`: 引入输入输出流库，提供 `cout` 对象用于输出。
- `using namespace std;`: 使用标准命名空间，简化代码。
- `int main()`:  主函数，程序的入口。
- `cout << "Hello, world!" << endl;`:  使用 `cout` 对象输出字符串 "Hello, world!"，`endl` 表示换行。
- `return 0;`:  返回值 0 表示程序正常结束。

**函数结构:**

```c++
返回类型 函数名称 (参数列表) {
  // 函数体
}
```

- 返回类型： 函数执行完成后返回的值的类型，例如 `int` 表示返回整型值。
- 函数名称： 函数的标识符，用于调用函数。
- 参数列表：  传递给函数的值，多个参数用逗号分隔。
- 函数体：  函数执行的代码块。


### 六、 C++ 数据类型 ......

- **基本数据类型:**

    - **int:** 整型，存储整数，通常占 4 个字节。
    - **char:** 字符型，存储单个字符，通常占 1 个字节。
    - **float:** 单精度浮点型，存储小数，通常占 4 个字节。
    - **bool:** 布尔类型，存储逻辑值 true 或 false。
    - **void:**  空类型，表示没有值，常用于函数返回值或参数。

- **算术运算符:**
    - `+` 加法
    - `-` 减法
    - `*` 乘法
    - `/` 除法
    - `%` 取余
    - `sizeof()`  获取数据类型或变量所占内存大小

- **整数除法:**  结果会被截断，只保留整数部分。
- **取余运算:**  获取两个整数相除后的余数。

- **其他基本数据类型:**
    - `short int` (简写为 `short`): 短整型
    - `long int` (简写为 `long`): 长整型
    - `long long int`(简写为 `long long`): 更长的整型
    
- **浮点型:**
    - `double`:  双精度浮点型，精度高于 `float`，通常占 8 个字节。
    - `long double`:  扩展精度浮点型，精度高于 `double`，具体大小取决于编译器。

- **无符号类型:**
    - 在数据类型前加 `unsigned` 关键字，表示该类型不存储符号位，只能存储非负数。
    - 例如：`unsigned int`,  `unsigned short`,  `unsigned long long`,  `unsigned char`。

**代码示例:**

```c++
#include <iostream>

using namespace std;

int main() {
  // 声明变量并赋值
  int age = 25;
  char initial = 'J';
  float height = 1.75;
  bool isStudent = true;

  // 打印变量值
  cout << "Age: " << age << endl;
  cout << "Initial: " << initial << endl;
  cout << "Height: " << height << endl;
  cout << "Is student? " << isStudent << endl;

  // 算术运算
  int a = 10;
  int b = 3;
  int sum = a + b;
  int difference = a - b;
  int product = a * b;
  int quotient = a / b; 
  int remainder = a % b;
  
  cout << "Sum: " << sum << endl;
  cout << "Difference: " << difference << endl;
  cout << "Product: " << product << endl;
  cout << "Quotient: " << quotient << endl; // 输出 3
  cout << "Remainder: " << remainder << endl; // 输出 1

  return 0;
}
```
