多文件编程

module2.cpp module2.h

main.cpp

---------------------------------------------------------

g++ -c module1.cpp -o module1.o
g++ -c module2.cpp -o module2.o
g++ -c main.cpp -o main.o

g++ -o test_module main.o module1.o module2.o // o 就是 object file

---------------------------------------------------------

更细化的过程：
g++ -E module1.cpp -> module1.i
g++ -E module2.cpp -> module2.i
g++ -E main.cpp    -> main.i


g++ -S module1.i -> module1.s
g++ -S module2.i -> module2.s
g++ -S main.cpp  -> main.s

as module1.s -> module1.o
as module2.s -> module2.o
as main.s -> main.o

ld main.o module1.o module2.o ... -> 可执行文件
